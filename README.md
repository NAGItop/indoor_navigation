## 🚀 快速开始

### 1. 配置高德地图（可选）

编辑 `amap-config.js`：

```js
window.APP_CONFIG = {
    amapKey: "你的高德Key",
    amapSecurityJsCode: "你的安全密钥jscode",
    amapCity: "海口"
};
```

> 获取方式：访问 [高德开放平台](https://lbs.amap.com/) 注册应用

### 2. 启动应用

直接打开 `index.html` 即可使用。

### 3. 演示账号

- 用户名：`demo`
- 密码：`123456`

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Esc` | 关闭弹窗 |
| `Ctrl/Cmd + E` | 紧急呼叫 |
| `← / →` | 步进导航上/下一步 |
| `Space` | 步进导航下一步 |
